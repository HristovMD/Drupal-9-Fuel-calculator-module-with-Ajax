INTRODUCTION
------------

 * This module provides you with block for a fuel calculator.

INSTALLATION
------------
 * Install as you would normally install a contributed drupal module and enable the Fuel Calculator module.

CONFIGURATION:
--------------
1. Visit /admin/structure/block and configure "Fuel Calculator" block.
2 .This block will enable Fuel Calculator.
